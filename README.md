# Academic-Emotional-Support-Portal
🎓 Academic Emotional Support Portal

A privacy-first, human-centered emotional intelligence platform built for academic environments, designed to capture emotional signals, enable early intervention, and empower counselors through intelligent tooling — without replacing human empathy with automation.

This project goes beyond traditional “support portals” by blending secure full-stack engineering, optional AI-assisted insights, and role-aware workflows to address emotional well-being in a scalable, ethical, and non-diagnostic way.

✨ Why This Is Different

Most platforms either:

Act as static feedback forms, or

Attempt automated mental health diagnosis

This system does neither.

Instead, it introduces a Human-in-the-Loop Emotional Support Model, where technology assists prioritization, visibility, and insight, while real counselors remain the decision-makers.

🚀 Core Features
🧑‍🎓 Student-Centric Emotional Check-Ins

Secure emotional check-ins with optional anonymity

Support request submission without stigma

Personal history view of past submissions and responses

🧠 Counselor Intelligence Dashboard

Role-based dashboard for reviewing emotional requests

Status tracking and response management

Priority visibility for high-risk emotional signals

📊 Emotion Analytics (Privacy-Preserved)

Aggregated mood trends across the platform

Visualization of request patterns and stress indicators

Institutional insights without exposing individual identities

⚠️ Smart Risk Awareness (Non-Diagnostic)

Keyword-based alerting for critical emotional indicators

Optional sentiment tagging to assist counselor triage

Early warning signals — not medical conclusions

🔐 Security & Ethics by Design

JWT-based authentication

Role-based access control (Student / Counselor / Admin)

No diagnosis, no automation of decisions, no data misuse

🧠 Innovation Highlights

Human-in-the-Loop Architecture – AI assists, humans decide

Optional Anonymity – Reduces emotional barriers to entry

Non-Diagnostic Intelligence – Ethical use of sentiment signals

Privacy-First Analytics – Insights without surveillance

Scalable Academic Deployment – Built for real institutions
